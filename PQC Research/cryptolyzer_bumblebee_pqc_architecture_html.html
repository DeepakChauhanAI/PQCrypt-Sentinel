PQC Architecture Proposal · May 2026

CryptoLyzer + Bumblebee
PQC Platform Architecture

A strategic proposal for combining protocol-level cryptographic analysis and dependency-level software inventory to accelerate the development of a professional post-quantum cryptographic discovery platform.

Executive Summary

Why This Combination Is Powerful

CryptoLyzer and Bumblebee operate at different visibility layers.

Together they provide:

Runtime cryptographic visibility + dependency inventory visibility.

This creates a strong foundation for building a professional PQC discovery and migration intelligence platform.

Tool Comparison

What Each Tool Contributes

CryptoLyzer

  • TLS handshake analysis
  • SSH cryptographic analysis
  • DNSSEC inspection
  • IKE / VPN discovery
  • Cipher suite detection
  • Host key analysis
  • JA3 / HASSH fingerprinting
  • Runtime cryptographic exposure visibility

Bumblebee

  • Dependency inventory
  • Package metadata scanning
  • Lockfile inspection
  • Supply-chain visibility
  • Developer environment analysis
  • SBOM-related discovery
  • Software package mapping
  • Cryptographic library discovery
Architectural Strength

Static + Runtime Correlation

The biggest advantage comes from correlating:

  • Installed cryptographic dependencies
  • Runtime cryptographic behavior

This enables the platform to distinguish between:

  • Available cryptographic capabilities
  • Actually negotiated cryptographic exposure

Platform Architecture

Recommended MVP Design

CryptoLyzer
Runtime protocol analysis, TLS negotiation inspection, SSH cryptographic discovery, and live network exposure visibility.
Bumblebee
Dependency inventory, software package analysis, metadata-driven discovery, and supply-chain visibility.
Normalization Layer
Converts heterogeneous scanner output into a canonical cryptographic asset schema.
CBOM Engine
Generates Cryptographic Bills of Materials for enterprise inventory and migration planning.
Risk Intelligence Engine
Performs PQC migration scoring, quantum risk analysis, crypto agility assessment, and remediation prioritization.

Final Conclusion

Long-Term Strategic Direction

CryptoLyzer + Bumblebee is not the final enterprise PQC platform.

However, it is:

An extremely strong foundation for building a professional cryptographic exposure management platform.

The opportunity is building:

The system of record for enterprise cryptographic exposure.
